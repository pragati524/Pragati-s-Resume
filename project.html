Overview

Here are some of the projects I've worked on during my journey as a Computer Science student. They showcase my skills in web development, machine learning, and programming. Each project helped me gain valuable insights and hands-on experience.

Project Summaries

1. Fake News Detection Website

Built a machine learning-based model to detect fake news, along with a user-friendly website.

  • Features: Homepage explaining fake news, links to reliable sources, fake news detection tool, and chatbot using Botpress.
  • Tech Stack: Python (ML), HTML, CSS, JavaScript
  • Outcome: Enhanced awareness about the spread of misinformation.

2. Student Productivity Tracker (Notion-based App)

Designed an app on Notion to help students track courses, assignments, and notes.

  • Features: Course tracking, assignment updates, and note organization.
  • Functionalities: Users can comment, add photos, and rate courses, as well as manage their learning progress.

3. BRRRGRRR Burger Website

Created a static burger-selling website during my 2nd year using basic front-end development skills.

  • Tech Stack: HTML, CSS, JavaScript
  • Outcome: Improved front-end skills and learned about responsive web design.

4. Currency Converter

Developed a Java-based currency converter as a 2nd-year project for quick and accurate conversions.

  • Features: Converts between Dollar, Rupee, Yen, Dirhams, and Pound.
  • Tech Stack: Java (Eclipse IDE)

5. Postman API Workshop Project

Participated in a hands-on Postman API workshop at NITK Surathkal, exploring API testing and integration.

  • Key Takeaway: Learned API testing concepts and best practices for software development.
